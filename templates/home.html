Yelp Restaurant Search in New York City

About this project

This project uses the Yelp API to collect 1000 records for restaurants in the New York City area. The data is stored using JSON files. To enable efficient range search and other functions, a KD-Tree is constructed based on the data.

Please select an option in the list to explore.

  1. View Restaurants in an Interactive Map
  2. Conduct a Single Condition Precise Search
  3. Conduct a Range Search
  4. See the Relation between Neighborhood Population and Restaurant Price and Rating
